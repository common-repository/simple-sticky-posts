Adds simple sticky posts feature to WordPress blogs.

== Installation ==

1. Upload `simple-sticky-post.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To set up a sticky post add a custom field named 'sticky' set to value '1' to one of your blog entries.
